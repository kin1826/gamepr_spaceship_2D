# **🚀 2D Spaceship Game**

## **📌 Introduction**

2D Spaceship Game is a midterm project for the Game Programming course. In this game, players control a spaceship to navigate through space, avoid and destroy obstacles (asteroids), and collect stars to progress through levels.

The main objective is to complete each level by filling a progress bar. Players gain progress by collecting stars, destroy asteroids to earn points, and collect items such as health packs and energy boosts.

## **🎮 Gameplay**

Control the spaceship with smooth and responsive movement
Shoot or dodge asteroids while collecting loot
Collect stars to complete level progression
Pick up power-ups (health, energy, buffs) to enhance abilities
Score system based on:
Number of destroyed asteroids
Progress achieved through collected stars

## **✨ Features**

🚀 Smooth and intuitive controls

💥 Asteroid destruction mechanics with visual effects

⭐ Progression system based on star collection

🎁 Power-up system (health, energy, upgrades)

🔊 Immersive sound effects and background music

📈 Scoring and level progression system

## **🛠️ Technologies Used**
Language: C#

Engine/Framework: Unity

Graphics: 2D Sprites

Audio: Sound effects and background music
